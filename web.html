For beginers the given list of commands can be used and also advanced linux commands can be used

  • mkdir "folder name" - Used to create folders
  • cd "path/folder name" - Used to change folder
  • ls - Used to list the contents in the folder
  • touch "file name" - Used to create files
  • cat "file name" - Used to view the content of the file
  • echo - Used to print the arguments
  • mv "source destination" - Used to move content to destination
  • cp "source destination" - Used to copy content to destination
  • pwd - Used to show the presnt working directory
  • rm "file name" - Used to remove file or empty folder
  • gedit "filename" - Used to open a file with gedit
  • vim "filename" - Used to open a file with vim editor
  • ifconfig - Used to configure the kernel-resident network nterfaces
  • ping - Used to check the status of a host or a network segment
